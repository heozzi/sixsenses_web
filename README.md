# 일취월장 프로젝트

프론트엔드 개발 진행

---
1. 로그인 페이지(/) - 김강빈
    - 닉네임을 입력 받는다 
    - 공백을 받을 시 Alert로 에러처리 
    - 중복 닉네임을 받을시 Alert로 에러처리 
    - 문제가 없을시 question 페이지로 전달
---
2. 문제 페이지 - 김강빈
    - 문제를 랜덤 추출하여 출제한다
    - 10문제가 될시 rank 페이지로 전달
---
3. 랭크 페이지 - 임서진
    - 상단에 입력 밭은 사용자의 정답수와 등수가 나타난다
    - 전체에서 1등 부터 5등이 리스트 형태로 화면에 나타난다.
    - 토글 스위치 클릭시 사용자 기준 윗 등수와 아래 등수가 보인다
